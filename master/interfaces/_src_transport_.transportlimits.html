Interface TransportLimits

A base class for a transport's configuration class. This is useful for transport implementations that use the readFrame and writeFrame functions.

desiredReadRate

desiredReadRate: Milliseconds

The desired minimum rate of incoming data from the connection peer. The actual expected rate is negotiated with the connection peer as the session is establishing. If no data is received during the interval then the connection is dead.
